# Mini Machines

Collection of small machines for esolangs made by me.

## Naming Scheme

We use N to specify the word size, M to specify the amount of words in memory and b to specify the base.

## Documentation

- N2M121b11 (2 undigits, 121 undigytes): [https://esolangs.org/wiki/2_undigits,_121_undigytes]()
